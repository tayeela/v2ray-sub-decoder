⚡ v2ray Sub Decoder
GitHub

Декодирование подписок vmess / vless / shadowsocks / trojan / hysteria2 в JSON, ссылки vless:// и готовый конфиг для xray. Всё считается прямо в браузере — данные никуда не отправляются (кроме самого запроса подписки по URL).

📁 Менеджер подписок

Облачное хранилище подписок на приватном GitHub Gist. Вход по токену с доступом только к Gist — токен хранится только в этом браузере и используется лишь для запросов к api.github.com. Введи тот же токен на другом устройстве — увидишь те же подписки.

Создать токен — scope gist уже будет отмечен, больше ничего включать не нужно. Срок действия — на твоё усмотрение (с истечением придётся войти заново).

    …или вставить содержимое подписки вручную

    Браузер блокирует прямые запросы к чужим серверам (CORS), поэтому режим «Авто» пробует прямой запрос и несколько публичных прокси, пока один не сработает. Прокси видят твою ссылку; для полной приватности (или если ничего не сработало) — открой ссылку подписки в новой вкладке, скопируй текст и вставь вручную.

    Время TCP/TLS-соединения до сервера из этого браузера (не скорость скачивания). UDP-ноды (hysteria2) могут не отвечать на такой тест.
    Работает офлайн как статический сайт · парсинг полностью на стороне браузера

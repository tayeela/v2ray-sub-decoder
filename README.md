# v2ray Sub Decoder

Веб-приложение для декодирования v2ray/xray подписок прямо в браузере.
Поддерживает ссылки `vmess://`, `vless://`, `ss://` (shadowsocks) и `trojan://`
и выдаёт результат в четырёх форматах:

- **JSON** — массив всех нод с разобранными полями;
- **vless** — ссылки `vless://` (только vless-ноды);
- **Ссылки** — нативные ссылки каждой ноды (round-trip);
- **Конфиг xray** — готовый `config.json` с локальным SOCKS/HTTP и outbounds.

Это статический сайт без бэкенда: весь разбор выполняется на стороне браузера,
поэтому его можно бесплатно хостить на GitHub Pages.

## Приватность

Когда вы вставляете содержимое подписки вручную — данные **никуда не уходят**,
всё считается локально в браузере.

Загрузка по URL из браузера обычно блокируется политикой CORS, поэтому по
умолчанию используется публичный CORS-прокси (`allorigins` / `corsproxy.io`).
Прокси видит вашу ссылку подписки. Для полной приватности используйте режим
«вставить содержимое вручную».

## Использование

1. Откройте сайт (см. ссылку GitHub Pages ниже).
2. Вставьте URL подписки и нажмите «Декодировать», либо раскройте
   «…вставить содержимое подписки вручную» и вставьте base64/список ссылок.
3. Переключайте вкладки JSON / vless / Ссылки / Конфиг xray.
4. Кнопки «Копировать» и «Скачать» — для сохранения результата.

### Запуск конфига xray

```bash
xray run -c config.json
```

Активна первая нода (xray по умолчанию использует первый outbound). Локальный
прокси: SOCKS5 `127.0.0.1:10808`, HTTP `127.0.0.1:10809`.

## Локальный запуск

Любой статический сервер, например:

```bash
python -m http.server 8765
# затем открыть http://localhost:8765
```

## Публикация на GitHub Pages

Settings → Pages → Source: **Deploy from a branch** → ветка `main`, папка `/ (root)`.
Через минуту сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.
